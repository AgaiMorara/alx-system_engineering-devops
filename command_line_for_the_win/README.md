How to share files with a remote server using SFTP:
==================================================
1. ) Collect the server credential logins... ( In this case from the sandbox sftp section... ) They are of the format: username@server_public_ip , and password
2.) navigate to the directory you want to put the files... In this case /root/alx-system_engineering-devops/command_line for the windows.
3.) navigate to the local directory with the files to be shared.
4.) Put the files to the remote directory.. ( put filename)...
